Here's a quick way to create a correlation network plot

June 8, 2018
R Visualization


Corrr package


From my previous post, I did mentioned about the use of network plot from ‘corrr’ package in visualizing correlation matrices. So, for this time, I’m going to show you how to create your own correlation network plot by using ‘corrr’ package in a quick way and also how do we actually intrepret it.






Installation


Download and install the ‘corrr’ and ‘magrittr’ R packages from CRAN for the first time before we can make use of the functions from it.


install.packages("corr")

install.packages("magrittr")






Network plot


library('corrr')

library('magrittr')

iris %>% correlate( ) %>% network_plot(min_cor = 0.3)



At first, we load the required R packages from the library(provided the packages are already installed). Next, we insert the sample dataset, ‘iris’ followed by correlate( ) function and then network_plot( ) function. min_cor specify the minimum correlation value(in absolute scale) to be plotted in a network plot and in this case, we specify all the correlation values among variables which below 0.3 and also -0.3 are eliminated which do not

Function Description

correlate( ) View all correlation value among variables in table(tibble format)

network_plot( ) Plot a correlation network plot

min_cor Minumum correlation value to be plotted in a network plot

%>% Forward pipe operator
